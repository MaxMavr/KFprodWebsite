## Коммиты
Коммит пишется по схеме: `что сделали [где сделали] комментарий подробнее, что сделали`

**Основные префиксы:**  
`add` — добавили  
`fix` — починили  
`refactor` — почистили  
`upd` — обновили  
`finish` — завершили (Не обязательно навсегда)  
`rewrite` — переписали текст

### Исключения
минорная чистка багов именуется `minor fix #{Номер фикса по порядку}`  
новая версия `chore {Номер версия}`

## Требования анимированным логотипам
Размер: не более 900 пикселей (по большей стороне)
Размер должен быть таким, чтобы линии в СВГ были заметными и имели достаточную толщину

Для автоматической очистки СВГ от загрязнений существует специальная программа, которую можно найти тут: `PythonProjects/ForDesign/Site/creating_demonstrations/svg/__init__`

## Требования к иконкам
Размер области: 24 пикселя  
Размер иконки: 12 пикселей (по большей стороне)  
Иконка должна быть выровнена по центру области  

Шаблон для иконок лежит тут: `./TEST/icons/icon-template.ai`
