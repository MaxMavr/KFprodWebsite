## Коммиты

Коммит пишется по схеме: `что сделали [где сделали] комментарий подробнее, что сделали`

**Основные префиксы:**
- `add` — добавили
- `fix` — починили
- `refactor` — почистили
- `rewrite` — переписали текст
- `upd` — обновили
- `chore` — добавили версии в РИДМИ
- `finish` — завершили (Не обязательно навсегда)

**Исключение:** минорная чистка багов именуется `minor fix #{Номер фикса по порядку}`

## Требования к иконкам

Размер области: 24 пикселя  
Размер иконки: 12 пикселей (по большей стороне)  
Иконка должна быть выровнена по центру области  

Шаблон для иконок лежит тут: `./TEST/icons/icon-template.ai`
