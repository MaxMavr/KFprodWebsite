Плакат Пиксели Дизеринг
Пост в тг

Последнее время меня очень привлекает эстетика пикселей, и дизеринг идеально вписывается в эту эстетику.

В обработке изображений дизеринг — это техника для создания иллюзии большей глубины цвета в изображениях с ограниченной цветовой палитрой.

И я давно хотел нарисовать плакат для самой любимой строчки из песни, а именно: «Кристаллизации искусства из беспорядка».

Эта фраза идеально описывает саму суть дизеринга — из хаоса пикселей и шума рождается упорядоченное изображение

Я не стал бы расставлять каждый пиксель вручную — написал скрипты на Питоне.

Есть несколько алгоритмов, как можно подмешивать шум в изображение. Одними из популярных являются «рандомный» и упорядоченный Байера 

def noise_dither(brightness: float) -> bool:
  threshold = random.random()
  return brightness > threshold
def pattern_dither_8x8(x: int, y: int, brightness: float) -> bool:
  threshold = BAYER_8x8[y % 8][x % 8]
  return brightness > threshold
